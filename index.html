Group 2 Exercise

This is the first exercise from group 2.We show a complete example to from raw data set to web-based visualisations. The data set can be found in this Google Spreadsheet.

1999-2003 sex ratio of Candidates

First checkout the sex of each party. You can see that they have different sex ratio .You can also click "Absolute Number of Votes" to find more information.

How?

  • Pivot table for winning candidates: pt Camp wining number
  • Pivot table for absolute number of votes: pt Camp abs vote
  • Generate code from pivot table result: st Camp to code

Mapping the parties

You can search online for GeoJSON files organised by the open data community. We use 2blam/HK-geojson in this case. Right side is the map to show votes winned by "Other" camp (non pro-establishment and non pan-democracy) in the year 2015.
